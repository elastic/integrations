# GCP Audit Logs OpenTelemetry Assets

This package contains Kibana assets for monitoring [GCP Audit Logs](https://docs.cloud.google.com/logging/docs/audit).

## Supported data sources

### EDOT Cloud Forwarder (ECF) for GCP

ECF is the simplest way to configure GCP log collection. For the setup instructions, check the [ECF documentation](https://www.elastic.co/docs/reference/opentelemetry/edot-cloud-forwarder/gcp) for full setup instructions.

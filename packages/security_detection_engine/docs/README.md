# Prebuilt Security Detection Rules

The detection rules package stores the prebuilt security rules for the Elastic Security
[detection engine](https://www.elastic.co/docs/solutions/security/detect-and-alert).

To download the rules, refer to [Install Elastic prebuilt rules](https://www.elastic.co/docs/solutions/security/detect-and-alert/install-prebuilt-rules#load-prebuilt-rules).
To update the rules to ensure they detect the latest threats and techniques, refer to
[Update Elastic prebuilt rules](https://www.elastic.co/docs/solutions/security/detect-and-alert/update-prebuilt-rules).
